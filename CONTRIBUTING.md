# Contributing to FlashScalper

Thank you for your interest in contributing to FlashScalper!

## Important Notes

- This is experimental software for educational purposes
- All contributions are welcome but subject to review
- Please ensure your code follows the existing style and patterns
- Add tests for new features when possible
- Update documentation as needed

## Development Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Copy `.env.example` to `.env` and configure
4. Build: `npm run build`
5. Run tests: `npm test`

## Code Style

- Use TypeScript with strict mode
- Follow existing code patterns
- Add JSDoc comments for public functions
- Use meaningful variable names
- Keep functions focused and small

## Pull Request Process

1. Create a feature branch
2. Make your changes
3. Ensure tests pass
4. Update documentation if needed
5. Submit a pull request with a clear description
